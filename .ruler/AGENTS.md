# AGENTS.md

## Project Description

This is **perplexity-mcp**, a Model Context Protocol (MCP) server that provides web search functionality using Perplexity AI's API. It enables AI assistants like Claude to search the web with recency filtering (day/week/month/year).

- **Language**: Python (uses `uv` for package management)
- **Main tool**: `perplexity_search_web` - searches the web via Perplexity's API
- **Installation**: Available via Smithery (`npx @smithery/cli install perplexity-mcp`) or `uvx perplexity-mcp`
